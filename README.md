# Murillo Oliveira Portfolio

Personal portfolio built with React, Vite and Tailwind CSS to present my backend development profile, selected projects, technical skills, journey and contact links.

## Overview

This website focuses on a clean, animated and responsive experience. It includes an intro animation, scroll reveal effects, mobile navigation and sections for professional background, projects, skills and contact information.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React

## Features

- Responsive layout for desktop and mobile
- Intro screen with typing animation
- Smooth scroll reveal effects
- Animated hero section
- Mobile navigation menu
- Project showcase
- Skills and professional journey sections
- Contact links

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
